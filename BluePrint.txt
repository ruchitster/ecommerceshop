# Multer Image Upload Flow

## Backend

1. User selects image
2. Multer uploads image to `/uploads`
3. Image path saved in MongoDB
4. Frontend renders image dynamically

## Product Upload API

```http
POST /api/products
```

FormData:

```js
image
name
price
category
subcategory
```

---

# JWT Authentication Flow

## Register

1. User registers
2. Password hashed using bcrypt
3. JWT token generated
4. Token stored in localStorage

## Login

1. User logs in
2. JWT token generated
3. Protected routes unlocked

---

# Razorpay Integration Flow

## Backend

1. Create Razorpay Order
2. Send order details to frontend
3. Verify payment signature

## Frontend

1. User clicks checkout
2. Razorpay popup opens
3. Payment completed
4. Order stored in database
